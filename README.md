# WindowsGSM.Citadel
WindowsGSM plugin that provides Citadel Dedicated server support!

# Requirements
WindowsGSM >= 1.21.0

# Installation
  1. Download the latest release
  2. Move Citadel.cs folder to plugins folder
  3. Click [RELOAD PLUGINS] button or restart WindowsGSM
